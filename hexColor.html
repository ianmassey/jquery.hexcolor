jQuery.hexColor Test Page

jQuery reports CSS color values as RGB numbers, like rgb(255,255,255). Sometimes it's useful to have these in hex instead.

hexColor hooks that up for you. It also gives you an array of multiple colors from a jQuery collection containing multiple elements which all have different colors, rather than just a string representing the first element in the collection's color.
